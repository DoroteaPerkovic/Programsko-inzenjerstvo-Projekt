# Contributing to StanPlan

Hvala što želite doprinijeti projektu **StanPlan**!   
Ovaj dokument sadrži smjernice kako možete pomoći razvoju aplikacije, bilo kodom, dokumentacijom ili prijedlozima.

## Kako prijaviti bug

1. Provjerite postoji li već sličan issue.
2. Ako ne postoji, otvorite novi issue s detaljnim opisom problema.
3. Dodajte screenshot ili primjer ako je moguće.

## Kako predložiti novu značajku

1. Otvorite issue i opišite svoju ideju.
2. Objasnite zašto bi značajka bila korisna.
3. Ako imate prijedlog rješenja, slobodno ga navedite.

## Doprinos kodu

1. Forkajte repozitorij i klonirajte ga lokalno.
2. Kreirajte novu granu za svoj rad:

   ```bash
   git checkout -b my-feature
   ```
3. Napravite promjene u skladu sa stilom projekta.
4. Pokrenite testove (ako postoje) i osigurajte da sve radi.
5. Pošaljite pull request s opisom promjena i linkom na relevantni issue.

## Dokumentacija

* Ako uređujete README ili dokumentaciju u `dokumentacija/` mapi, pazite na jasnoću i jezik koji je razumljiv svima.

## Kodni stil

* Slijedite postojeći stil koda u `src/` i `backend/`.
* Komentirajte kompleksnije dijelove koda radi lakšeg razumijevanja.

## Komunikacija

* Budite pristojni i konstruktivni.
* Poštujte mišljenja drugih članova tima.

Hvala što doprinosite projektu i pomažete StanPlanu da postane bolji!
